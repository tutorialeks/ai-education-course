## 1. Learning & Understanding

### 1.1 Explain a Concept
```
Explain [CONCEPT] in simple terms.

My background: [WHAT YOU ALREADY KNOW]
Level of detail I need: [BRIEF OVERVIEW / MODERATE / IN-DEPTH]

Include:
- Core definition in one sentence
- A real-world analogy
- A simple example
- Common misconceptions to avoid
```

### 1.2 Compare and Contrast
```
Compare [THING A] and [THING B].

Create a table showing:
- Key similarities
- Key differences
- When to use each
- Common confusions between them
```

### 1.3 Break Down Complexity
```
I'm overwhelmed by [COMPLEX TOPIC].

Break it down into:
1. The 3-5 core components I must understand first
2. How they connect to each other
3. What order to learn them in
4. What I can safely ignore for now
```

### 1.4 Multiple Explanations
```
Explain [CONCEPT] in three different ways:
1. A technical explanation
2. An everyday analogy
3. A visual description (describe what a diagram would look like)

I'll tell you which one clicked for me.
```

### 1.5 Depth Levels
```
Explain [CONCEPT] at three levels:

Level 1: ELI5 (Explain Like I'm 5)
Level 2: High school student
Level 3: University-level detail

Label each level clearly.
```

### 1.6 "Why Does This Matter?"
```
I'm learning [TOPIC] but don't see why it matters.

Explain:
1. Real-world applications
2. How it connects to things I care about: [YOUR INTERESTS]
3. What problems it solves
4. What would be impossible without it
```

### 1.7 Prerequisites Check
```
I want to learn [ADVANCED TOPIC].

What prerequisite knowledge do I need?
For each prerequisite:
- Why it's necessary
- How to tell if I know it well enough
- Quick resource to fill the gap if needed
```

## 2. Study Planning

### 2.1 Curriculum Generator
```
Create a [X]-week learning plan for [TOPIC].

My details:
- Current level: [BEGINNER/INTERMEDIATE/ADVANCED]
- Goal: [SPECIFIC OUTCOME]
- Time available: [X] hours per week
- Learning style: [READING/VIDEO/HANDS-ON/MIXED]

For each week include:
1. Learning objectives
2. Topics to cover
3. Recommended free resources
4. Practice activities
5. Self-check: How to verify I understood it
```

### 2.2 Self-Assessment
```
Test my knowledge of [TOPIC].

Ask me 10 questions ranging from basic to advanced.
Wait for my answer before asking the next.

After all questions:
- Estimate my level
- Identify specific gaps
- Recommend what to study next
```

### 2.3 SMART Goal Setting
```
Help me set SMART learning goals for [TOPIC].

Timeframe: [X WEEKS/MONTHS]
My motivation: [WHY I'M LEARNING THIS]

Create 3-5 goals that are:
- Specific (exactly what I'll be able to do)
- Measurable (how I'll know I achieved it)
- Achievable (realistic for my timeframe)
- Relevant (connected to my motivation)
- Time-bound (clear deadline)
```

### 2.4 Study Schedule
```
Create a weekly study schedule.

Available time:
- Weekday mornings: [X hours]
- Weekday evenings: [Y hours]
- Weekends: [Z hours]

I work best for [30/60/90] minute sessions.
Topics to cover: [LIST]
Deadline: [DATE]

Balance new learning with review sessions.
```

### 2.5 Priority Matrix
```
I need to learn these topics: [LIST]
Available time: [X HOURS TOTAL]
Goal: [EXAM/PROJECT/JOB]

Create a priority matrix:
- MUST KNOW (essential, high-impact)
- SHOULD KNOW (important but not critical)
- NICE TO KNOW (if time permits)
- SKIP (low value for my goal)

Explain your reasoning.
```

## 3. Practice & Testing

### 3.1 Practice Test Generator
```
Create a practice test on [TOPIC].

Format: [MULTIPLE CHOICE / SHORT ANSWER / ESSAY / MIXED]
Number of questions: [X]
Difficulty: [EASY / MIXED / HARD]
Time limit: [X MINUTES]

Include an answer key with explanations.
Keep answers hidden until I ask for them.
```

### 3.2 Bloom's Taxonomy Questions
```
Test my understanding of [TOPIC] at all cognitive levels:

1. REMEMBER: Recall facts
2. UNDERSTAND: Explain in own words
3. APPLY: Use in new situation
4. ANALYZE: Break down, find patterns
5. EVALUATE: Judge, defend position
6. CREATE: Generate something new

2 questions per level.
```

### 3.3 Weak Point Drill
```
My weak areas in [SUBJECT] are: [LIST]

Create targeted practice:
- 3 questions per weak area
- Start easy, increase difficulty
- Include hints if I'm stuck
- Explain the underlying concept when I get it wrong
```

### 3.4 Spaced Repetition Review
```
I learned these concepts [X DAYS] ago: [LIST]

Test my retention:
- Ask one question per concept
- Don't give hints
- After I answer, tell me which need more review
- For forgotten ones, give a quick refresher
```

### 3.5 Exam Simulation
```
Simulate a [SUBJECT] exam.

Exam format: [DESCRIBE]
Time: [X MINUTES]
Topics: [LIST]

Give me the exam. I'll work through it and tell you when done.
Then grade me and give detailed feedback on:
- Accuracy
- Time management advice
- Areas to review
```

### 3.6 Quick Quiz
```
5-question quick quiz on [TOPIC].
I have 5 minutes. Keep questions concise.
Grade immediately after each answer.
Give final score at the end.
```

## 4. Language Learning

### 4.1 Conversation Partner
```
You are my [LANGUAGE] tutor.

My level: [A1/A2/B1/B2/C1/C2]
Rules:
- Speak only in [LANGUAGE]
- Match my level
- Correct mistakes gently
- Keep responses conversational (2-3 sentences)

Start a casual conversation about [TOPIC].
```

### 4.2 Role-Play Scenario
```
Let's practice a scenario in [LANGUAGE].

Scenario: [RESTAURANT / INTERVIEW / SHOPPING / etc.]
You play: [ROLE]
I play: [ROLE]
My level: [LEVEL]

Stay in character. After we finish, give feedback on my language.
```

### 4.3 Grammar Explanation
```
I wrote: "[SENTENCE WITH ERROR]"

1. What's wrong?
2. Correct version
3. Explain the rule simply
4. 3 similar examples
5. Memory trick for this rule
```

### 4.4 Vocabulary Builder
```
Give me 15 essential words for [TOPIC] in [LANGUAGE].

For each word:
| Word | Pronunciation | Meaning | Example Sentence |

Include common collocations.
End with one useful idiom related to this topic.
```

### 4.5 Reading Practice
```
Write a short text in [LANGUAGE] at [LEVEL].

- 200-300 words
- Topic: [TOPIC]
- Mark 5-10 new vocabulary words with *

After the text:
1. List vocabulary with definitions
2. 3 comprehension questions
```

### 4.6 Translation Exercise
```
Give me 5 sentences in [LANGUAGE A] to translate to [LANGUAGE B].
Mix difficulty levels.

After each attempt, show me:
1. Correct translation
2. Alternative acceptable versions
3. Common mistakes learners make
```

## 5. Programming

### 5.1 Concept Explainer
```
Explain [PROGRAMMING CONCEPT] for a beginner.

- No jargon without defining it
- Use a real-world analogy
- Show the simplest possible code example
- Explain each line
```

### 5.2 Debugging Help
```
I have a bug.

What I expect: [EXPECTED]
What happens: [ACTUAL]
Error message: [ERROR]

Code:
[CODE]

Don't fix it for me. Instead:
1. What's likely wrong?
2. How can I find the exact problem?
3. What should I try first?
```

### 5.3 Code Review
```
Review my code for:
1. Bugs or errors
2. Readability
3. Performance
4. Security issues
5. Best practices

Be critical‚ÄîI want to learn.

[CODE]
```

### 5.4 Project Guidance
```
I'm building [PROJECT] in [LANGUAGE].
My skill level: [LEVEL]

Don't write the code. Instead:
1. Break it into 5-7 steps
2. For each step, what should I figure out?
3. What to Google if stuck
4. Common pitfalls to avoid
```

### 5.5 Error Message Decoder
```
Explain this error in plain English:

[ERROR MESSAGE]

1. What does each part mean?
2. Most common causes
3. How to find where it's happening
4. What to try first
```

### 5.6 Pattern/Concept Teaching
```
Teach me [PATTERN/CONCEPT] in [LANGUAGE].

1. When/why would I use this?
2. Simple example
3. Real-world use case
4. Common mistakes
5. How do I know if I'm overusing it?
```

## 6. Writing & Content

### 6.1 Outline Generator
```
Create an outline for [CONTENT TYPE] about [TOPIC].

Audience: [WHO]
Length: [WORDS/PAGES/MINUTES]
Purpose: [INFORM/PERSUADE/ENTERTAIN]

Include:
- Attention-grabbing opening
- 3-5 main points with sub-points
- Transitions
- Strong conclusion
```

### 6.2 Feedback Request
```
Give feedback on my writing:

[YOUR TEXT]

Evaluate:
1. Clarity: Is the message clear?
2. Structure: Is it well organized?
3. Evidence: Are claims supported?
4. Engagement: Is it interesting?
5. Grammar: Any errors?

Be specific and constructive.
```

### 6.3 Improve My Draft
```
Help me improve this draft:

[YOUR DRAFT]

Don't rewrite it. Instead:
1. Identify 3 strengths to keep
2. Identify 3 weaknesses to fix
3. For each weakness, suggest how to improve
4. Any structural changes needed?
```

### 6.4 Presentation Outline
```
Create a [X]-minute presentation outline on [TOPIC].

Audience: [WHO]
Their knowledge level: [LEVEL]
Goal: [WHAT THEY SHOULD KNOW/DO AFTER]

For each slide suggest:
- Headline (key takeaway)
- 2-3 bullet points max
- Visual suggestion
```

### 6.5 Worksheet Creator
```
Create a worksheet on [TOPIC] for [LEVEL].

Include:
- Clear instructions
- 2-3 warm-up questions
- 5-7 main questions (increasing difficulty)
- 1 challenge question
- Answer key with explanations
```

## 7. Research

### 7.1 Topic Overview
```
Give me a comprehensive overview of [TOPIC].

Include:
1. Definition and core concepts
2. Historical background (brief)
3. Current state of knowledge
4. Key debates or controversies
5. Practical applications
6. Further reading suggestions
```

### 7.2 Source Finder
```
I'm researching [TOPIC].

Suggest:
- 3-5 key search terms to use
- Types of sources to look for
- Authoritative sources in this field
- Red flags for unreliable sources
- How to evaluate conflicting information
```

### 7.3 Argument Analysis
```
Analyze this argument:

[ARGUMENT]

1. Identify the main claim
2. What evidence is provided?
3. What assumptions are made?
4. What counterarguments exist?
5. How strong is the argument overall?
```

### 7.4 Literature Summary
```
Summarize the main points of this text:

[TEXT]

Include:
- Main argument/thesis
- Key supporting points
- Methods used (if applicable)
- Conclusions
- Limitations
- How this relates to [YOUR TOPIC]
```

### 7.5 Research Question Refinement
```
My initial research question: [QUESTION]

Help me refine it to be:
1. More specific
2. Answerable with available methods
3. Interesting/significant
4. Neither too broad nor too narrow

Suggest 3 alternative formulations.
```

## 8. Memorization

### 8.1 Mnemonic Creator
```
Create mnemonics to remember:

[LIST OF ITEMS]

For each, create:
- An acronym, rhyme, or visual association
- Explanation of the connection
- Make them vivid, funny, or absurd
```

### 8.2 Flashcard Generator
```
Create 10 flashcards for [TOPIC].

Format:
FRONT: [Term/Question]
BACK: [Definition/Answer]
MEMORY HOOK: [Mnemonic or association]

Make example sentences memorable‚Äîweird/funny works best.
```

### 8.3 Memory Palace Guide
```
Help me create a memory palace for [LIST].

I'll use this location: [YOUR FAMILIAR PLACE]

For each item:
1. Assign it to a specific spot
2. Create a vivid, interactive image
3. Link the image to the location
4. Create a story connecting them all
```

### 8.4 Active Recall Test
```
Test my recall on [TOPIC].

Ask me 5 questions. No hints.
After each answer:
- Correct? Great, move on
- Wrong? Let me try again before revealing answer

This is harder than re-reading, but more effective.
```

### 8.5 Spaced Repetition Schedule
```
I need to memorize [AMOUNT] by [DATE].

Create a spaced repetition schedule:
- Day 1: First learning
- Days for review: 1, 3, 7, 14, 30

Include:
- What to do at each review
- How to handle forgotten items
- Daily time estimate
```

## 9. Teaching & Explaining

### 9.1 Analogy Generator
```
Create 3 analogies for [CONCEPT].

For each:
1. The comparison
2. How it maps to the concept
3. Where the analogy breaks down
4. Best audience for this analogy
```

### 9.2 Explanation at Multiple Levels
```
Explain [CONCEPT] for three audiences:

1. A child (age 10)
2. A smart adult with no background
3. Someone with basic knowledge who needs details

Label each clearly.
```

### 9.3 Anticipate Confusion
```
I'm teaching [TOPIC] to [AUDIENCE].

Predict:
1. 5 questions they'll likely ask
2. 3 misconceptions they'll have
3. Where they'll get stuck
4. What prerequisites might be missing

For each, suggest how to address it.
```

### 9.4 Examples Generator
```
Give me 5 examples of [CONCEPT]:

1. Simple, classic example
2. Real-world, relatable example
3. Surprising/counterintuitive example
4. Non-example (shows what it's NOT)
5. Example they could recreate themselves
```

### 9.5 Differentiated Explanation
```
Adapt this explanation for different learners:

Original: [YOUR EXPLANATION]

Create versions for:
1. Struggling learner (more scaffolding)
2. Typical learner (grade level)
3. Advanced learner (deeper/harder)
```

## 10. Metacognition & Review

### 10.1 Learning Reflection
```
Help me reflect on what I learned today.

Topics studied: [LIST]
Time spent: [X HOURS]

Ask me:
1. What was the most important concept?
2. What's still confusing?
3. How does this connect to what I knew before?
4. How might I use this?
5. What should I review tomorrow?
```

### 10.2 Understanding Check
```
I think I understand [CONCEPT].

Test whether I truly understand by:
1. Asking me to explain it without jargon
2. Giving me a novel scenario to apply it
3. Asking me to connect it to [DIFFERENT TOPIC]
4. Identifying any gaps in my explanation
```

### 10.3 Progress Review
```
Weekly learning review.

What I planned: [ORIGINAL PLAN]
What I completed: [ACTUAL]
Challenges: [DESCRIBE]

Help me:
1. Assess if I'm on track
2. Identify what slowed me down
3. Adjust next week's plan
4. Celebrate progress made
```

### 10.4 Knowledge Transfer
```
I learned [CONCEPT] in [CONTEXT A].

Help me apply it to [CONTEXT B]:
1. What transfers directly?
2. What needs adaptation?
3. What's different about the new context?
4. Give me a practice scenario in the new context.
```

### 10.5 Skill Audit
```
Audit my skills in [AREA].

Ask me about:
1. Specific capabilities
2. Recent applications
3. Confidence levels
4. Areas wanting improvement

Then give me:
- Honest assessment of my level
- Priority areas to develop
- Concrete next steps
```

## Quick Reference

### When Stuck
```
I'm stuck on [PROBLEM].
I've tried: [ATTEMPTS]
I don't understand: [SPECIFIC CONFUSION]
Point me in the right direction without giving the answer.
```

### When Overwhelmed
```
I'm overwhelmed by [TOPIC/TASK].
Help me:
1. Break it into smaller pieces
2. Identify what to do first
3. Set a realistic next step I can do in 15 minutes
```

### When Lost
```
I've been studying [TOPIC] but feel lost.
Help me:
1. Identify what I actually understand
2. Find where my understanding breaks down
3. Create a plan to rebuild from there
```
